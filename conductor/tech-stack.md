# Tech Stack: 2026-YTT Dossier

## Core Languages & Runtimes
- **Primary Language:** TypeScript (for type-safe development and scripting).
- **Runtime:** Node.js (v22+ recommended).
- **Package Manager:** `pnpm` (required for resource efficiency).

## Frontend (Dossier Website)
- **Framework:** React.
- **Build Tool:** Vite (required for speed and efficiency).
- **UI/Styling:** Material Design principles (using libraries like MUI or similar).
- **Content Rendering:** Markdown-to-React (for dynamic rendering of the dossier source files).

## Automation & Data Processing
- **Scripting:** TypeScript/Node.js for all automation tasks (e.g., timeline processing, PDF generation).
- **Analysis:** Gemini CLI and Google Jules 2.0.

## Infrastructure & Version Control
- **Version Control:** Git.
- **Hosting:** GitHub (private repository).
